CentOS 安装 Node.js、npm、htop、pm2

Nix_____
2017-05-21
技术整理
bash

1. 下载node.js

$ wget http://nodejs.org/dist/v0.12.0/node-v0.12.0.tar.gz

2. 解压node.js

$ tar xvf node-v0.12.0.tar.gz

3. 编译及安装

$ make && make install

4. 查看node版本

$ node -v

1. 安装npm

$ sudo yum install npm

2.查看npm版本

$ npm -v

1. 安装 htop

$ sudo yum install htop

2. 安装pm2

$ npm install -g pm2
